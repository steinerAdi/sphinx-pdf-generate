Admonitions
-----------



Formulas
--------

A formula can be inline like: `:math:\`\frac{3}{4}\\,\\text{m}\`.` results in :math:`\frac{3}{4}\,\text{m}`.

A centered extended formula can be used with the math directive:

.. math::

    a^{2} + b^{2} = c^{2}
